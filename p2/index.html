Target Platform Definition DSL and Generator - Releases Repository

This tool is released in several versions, one for each Eclipse's version. This is an Eclipse Update site/p2 repository providing all the releases of these versions. You should access it from Eclipse by using the following URL:

http://mbarbero.github.com/fr.obeo.releng.targetplatform/p2/

Repositories targeting a specific Eclipse version can be a accessed with the following URLs:

  • http://mbarbero.github.com/fr.obeo.releng.targetplatform/p2/3.6 for Eclipse 3.6 (Helios)
  • http://mbarbero.github.com/fr.obeo.releng.targetplatform/p2/3.7 for Eclipse 3.7 (Indigo)
  • http://mbarbero.github.com/fr.obeo.releng.targetplatform/p2/3.8 for Eclipse 3.8/4.2 (Juno) and later
